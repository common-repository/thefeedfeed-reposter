== Description ==

Thefeedfeed Reposter send your new posts to thefeedfeed.com blog

Major features in Thefeedfeed Reposter include:

* Automatically send your post to thefeedfeed.com when you publish or update post


== Installation ==

Upload the Thefeedfeed Reposter plugin to your blog, Activate it.

You're done!
